# MoodSnap
A beautiful, engaging daily mood tracking app that uses AI to analyze your selfies and journal entries to help you understand your emotional patterns. Take a daily selfie, write a short journal entry, and let our AI give you personalized insights and mood trends over time.

Key Features:
- Selfie-based mood detection using facial expression analysis
- Daily journaling with sentiment analysis
- Beautiful mood visualization charts
- Personalized insights and patterns
- Streaks and achievements to encourage daily use

Tech Stack:
- HTML/CSS/JavaScript frontend
- Face-api.js for facial emotion detection
- Chart.js for visualizations
- Local Storage for data persistence
